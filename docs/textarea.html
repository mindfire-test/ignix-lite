Usage

Use native textarea inside label.


<label>
  Message
  <textarea rows="3"></textarea>
</label>

Attributes

Rows


<textarea rows="5"></textarea>

Required


<textarea rows="3" required></textarea>

States

Default


<textarea rows="3"></textarea>

Disabled


<textarea rows="3" disabled></textarea>

Invalid


<textarea rows="3" aria-invalid="true"></textarea>
