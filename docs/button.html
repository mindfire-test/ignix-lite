Usage

Use the native button element with data-intent.


<button data-intent="primary">Save</button>

Intent Variants

Primary


<button data-intent="primary">Primary</button>

Danger


<button data-intent="danger">Danger</button>

Warning


<button data-intent="warning">Warning</button>

Success


<button data-intent="success">Success</button>

Neutral


<button data-intent="neutral">Neutral</button>

Ghost


<button data-intent="ghost">Ghost</button>

States

Default


<button data-intent="primary">Default</button>

Disabled


<button data-intent="primary" disabled>Disabled</button>

Loading


<button data-intent="primary" aria-busy="true">Loading</button>
